# 9983 Portal

## Overview
The 9983 Portal is a secure, feature-rich web application designed to manage operations and enhance situational awareness. The application incorporates multiple advanced functionalities, including secure login, map integration, computer-aided dispatch (CAD), AI-driven report generation, image steganography, and seamless integration with IPFS. It is built with modern web technologies to ensure a responsive and interactive user experience.

---

## Features

### Core Functionalities
- **Splash Screen**: Engages users with an animated welcome screen.
- **Secure Login**: Supports username/password authentication and OAuth (Google and Microsoft).
- **Dynamic Map**:
  - Displays markers for crime data and various organizational locations.
  - Integrates with KrakenSDR for real-time signal tracking.
- **AI-Powered Report Generator**: Allows law enforcement officers to create detailed incident reports using OpenAI's API.
- **Computer-Aided Dispatch (CAD)**: Logs incidents, assigns units, and exports logs.
- **Image Steganography**: Enables users to encrypt and decrypt messages in images.
- **TAK Server Configuration**: Configures FreeTAKServer settings.
- **IPFS Integration**: Uploads files to IPFS for decentralized storage.

### Additional Features
- **Status Window**: Displays real-time user details, including IP address, location, browser information, and screen resolution.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Multiple Modals**: Separate modal windows for distinct operations.
- **Real-Time Data Fetching**: Retrieves data dynamically for maps, user metrics, and more.

## Technologies Used

### Frontend
- **HTML5**: Semantic and modern markup.
- **CSS3**: Custom styles for a dark and light theme.
- **JavaScript**: Includes modular scripts for dynamic interactions.

### Libraries and Frameworks
- [Leaflet](https://leafletjs.com/): Interactive maps.
- [CryptoJS](https://cryptojs.gitbook.io/): Secure hashing for passwords.
- [Helia](https://github.com/ipfs/helia): IPFS integration for decentralized storage.

### APIs
- [OpenAI](https://openai.com/): For AI-powered report generation.
- [IPAPI](https://ipapi.co/): Geolocation based on IP address.
- [Pinata](https://pinata.cloud/): IPFS file pinning.
